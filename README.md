### 已测试可正常使用功能：

网卡，声卡，无线网卡无解，独显硬解正常，可登入ID



### 注意事项
* 安装后删掉config文件，将名为（安装后的config）改名为config，即可驱动显卡
* 自行更换三码
* BIOS设置：安全启动，CSM，快速启动关掉，Bios需关闭Resizable BAR
